# MakeCode-like Block Programming Demo

このプロジェクトは、MakeCodeやQUREOのようなブロックプログラミングUIをWebブラウザ上で体験できるデモサイトです。
[Blockly](https://developers.google.com/blockly) を活用し、HTML、CSS、JavaScriptのみの静的な構成で実装しています。

## 特徴
- 実行機能や保存機能を含まない、純粋なUIデモです。
- Scratch風の丸みを帯びたブロックスタイル（Zelosレンダラー）を採用しています。

## 用意されているブロック
左側のツールボックス（基本カテゴリ）から以下の7つの専用ブロックを使用できます。

- **イベント（黄緑色）**
  - チャットコマンド `[ ]` を入力した時
- **ループ（オレンジ色）**
  - くりかえし `[ ]` 回
- **アクション（ピンク色）**
  - エージェントを `[方向]` に `[数]` ブロック移動させる
  - エージェントの向きを `[方向]` に変える
  - エージェントに `[方向]` を破壊させる
  - エージェントに `[方向]` に置かせる
  - エージェントのスロット番号 `[数]` を有効にする

## ローカル環境での動かし方

Node.jsがインストールされている環境であれば、一時的なサーバーを立ち上げてプレビュー可能です。

```bash
# 依存関係のインストール
npm install

# ローカルサーバー起動
npm start
```

## 自動デプロイ
このリポジトリは GitHub Actions を用いた GitHub Pages の自動公開に対応しています。
`main` ブランチに Push が行われるたびに、あらかじめ設定した Workflow (`.github/workflows/deploy.yml`) によって自動デプロイが走ります。
